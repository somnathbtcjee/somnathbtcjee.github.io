Research Internships
Oct'23-Jan'24
Project on Witness for ABP-PIT via non-commutativity under prof. Partha Mukhopadhyay and prof Iddo Tzameret
In this project, I studied upper bounds and lower bounds for the proof-size of non commutative ABPs (which are commutatively zero) using commutators as axioms.
Summer 2023
Worked on Black-box approximate algorithm for Edmond problem under prof. Partha Mukhopadhyay and prof Abhranil Chatterjee
In this project I tried to ﬁnd black box subexponential approximate algorithm for edmonds’ problem using Bläser-Jindal-Pandey idea. I also read about algorithms for matroids using edmonds’ problem (commutative and non-commutative both) algorithms.
Oct'22-Jan'24
Worked on The conditional optimality of Ryser formula: Algebraic version of Strong ETH with prof. Markus Bläser, Pranjal Dutta and Saswata Mukherjee
In this project I learned about parameterized complexity and ﬁne grained complexity and their relation with algebraic complexity. We ﬁnally achieve exponential lower bound in parameterized valiant classes from tau-conjecture. This project leads to the paper Exponential Lowerbounds via Exponential sums.
